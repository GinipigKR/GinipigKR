### 2D Action Roguelite Game

Unity 기반 2D 액션 로그라이트 게임입니다.  
뱀서라이크 전투 구조를 바탕으로 아이템 성장, 스킬 조합, 카드 선택 시스템을 구현하고 있습니다.

**담당한 주요 기능**

- 아이템 획득 및 레벨업 시스템
- 카드 선택 UI 및 후보 생성 로직
- 오라 / 투사체 / 궤도형 스킬 구현
- 스킬 체인 시스템 구조 설계
- 조합 아이템 등장 조건 처리
